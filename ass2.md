# Todo Application README

## Project Overview

The Todo Application is a simple task management system that helps users organize and track their daily activities. Users can create, update, delete, and view tasks efficiently.

---

## Features

* Add new tasks
* View all tasks
* Update existing tasks
* Delete tasks
* Mark tasks as completed
* User-friendly interface

---

## Installation Guide

1. Clone the repository.
2. Navigate to the project directory.
3. Install the required dependencies.
4. Run the application.

```bash
git clone https://github.com/saiprasanna461/todo-application.git
cd todo-application
```

---

## Usage Instructions

1. Start the application.
2. Add tasks using the add command.
3. View tasks using the list command.
4. Update or delete tasks as needed.
5. Mark tasks as completed after finishing them.

---

## Sample Commands

```bash
todo add "Complete Markdown Assignment"
todo list
todo update 1 "Submit Assignment"
todo complete 1
todo delete 1
```
